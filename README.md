# EventSpace

Private, temporary rooms for shared events. EventSpace combines chat, a collaborative photo board, and an itinerary; rooms freeze into read-only archives at their end time.

- `docs/product-specification.md` — confirmed first-release product specification

- `docs/technical-architecture.md` — approved implementation architecture
- `docs/implementation-plan.md` — ordered delivery plan and acceptance gates
- `docs/requirements-baseline.md` — detailed decision log
